# robosys2024
ロボットシステム学授業用

# kadai
１人でじゃんけんとあっちむいてほいをするコマンド

# プログラミングの使用方法
- robosys2024ディレクトリに移動した上で、
`./kadai`で実行出来ます。

```
cd robosys2024
```
```
./kadai
```

# 遊び方
```
じゃんけん
1:グー　2:チョキ　3:パー
何を出しますか？
```
```
あっちむいてほい
1:←　2:↑　3:↓　4:→
どっちを指しますか？
```
のような形の説明に沿って、
じゃんけんで出す手や、あっちむいてほいで向く方向を数字で入力します。
