# kadai
１人でじゃんけんとあっちむいてほいをするコマンド

# 使用方法
- robosys2024ディレクトリに移動した上で、
`./kadai`で実行出来ます。

```
cd robosys2024
```
```
./kadai
```

```
じゃんけん
1:グー　2:チョキ　3:パー
何を出しますか？
```
